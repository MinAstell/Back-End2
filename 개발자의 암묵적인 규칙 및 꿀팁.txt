개발자 암묵적인 규칙 및 꿀팁

1. 패키지 이름은 소문자

2. 클래스 앞문자는 대문자로 시작

3. 메소드 이름은 동사로 (~하다)

4. 변수이름은 명사로

5. bool 타입은 변수이름 앞에 is or has를 사용

6. 상수(숫자)값은 변수이름을 모두 대문자로 함 EX). double PI = 3.14;

7. println 단축키  >>  syso를 치고 ctrl + spacebar + enter 

8. String은 클래스(타입)임 

9. 자바 추천메뉴? 보고싶다면, ctrl + spacebar 

10. if문법을 써야한다면, else는 최대한 쓰지말자(깔끔해지고, 콜백지옥 탈출가능)

11. boolean을 flag 값이라고 부르기도 함 

12. boolean isLogin = true; 
    if(isLogin == true) (X)  /  if(isLogin) (O) 중요!!

13. 스태틱 메소드에는 스태틱 메소드만 올 수 있음!! (중요)

14. 리턴타입이 있으면 return을 써야하며, void는 리턴타입이 없으므로 안써도 됨 

15. 메소드에 파라미터가 있다면 호출할 때 파라미터값도 써줘야 함

16. main 메소드 단축키  >>  main 치고 ctrl + spacebar + enter 

17. 메소드는 파라미터에 배열도 받을 수 있음

18. 리턴타입에 배열도 올 수 있음
